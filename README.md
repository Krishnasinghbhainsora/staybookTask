# Collaborative To-Do App

## Overview
This is a collaborative to-do list application built with React for the frontend and Node.js for the backend. It allows users to add, update, delete, and share tasks with others.

## Features
- User authentication
- Add, edit, and delete tasks
- Filter tasks by status (All, Pending, Completed)
- Share tasks with others via email

## Prerequisites
- Node.js installed
- A backend server running (use the appropriate API base URL in the app)

## Installation

### Backend
1. Run the following commands:
  ```
   npm install
   npm start
   ```
2. The backend server will start, usually at `http://localhost:8080`

### Frontend

1. Run the following commands:
   ```
   npm install
   npm run dev
   ```
2. The frontend application will start, usually at `http://localhost:5173`.

## Running the App
1. Start the backend server using `npm start` in the backend folder.
2. Start the frontend app using `npm run dev` in the frontend folder.


## Notes
- Make sure the backend server is running before using the app.
- Update the `API_BASE_URL` in the frontend code if the backend URL is different.
